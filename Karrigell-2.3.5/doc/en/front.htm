 Karrigell is a simple web programming solution, written in Python. It has been designed to be really simple to use : integrated web server and data base (PyDbLite), easy access to environment data and form fields, yet full-featured and powerful : Python script execution in the same process as the server, server pages including html and Python code (similar to PHP, JSP, ASP), easy handling of authentication, session management, localization features, etc

Karrigell can also work with external web servers : Apache, Xitami and LightTPD are currently supported. All current databases (sqlite, MySql, PostGresQL, etc) can be used with the corresponding Python API's

A set of demo files is provided, including a Wiki server, a forum, a blog engine and a portal with news publishing and personalization
