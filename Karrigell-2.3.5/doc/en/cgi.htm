9. CGI scripts

New in version 2.3.1

If you have developed CGI scripts on another environment, you can run them with Karrigell without modification. Warning : in version 2.3.1 this only works with the asynchronous web server ; support for other servers will be added later

All you have to do is put them in specific directories. By default, use the folder webapps/cgi-bin ; other paths can be specified in the configuration file, section Directories, option cgi
