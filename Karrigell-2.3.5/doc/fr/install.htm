2. Installation de Karrigell

Il faut d'abord installer Python 2.3 ou au-delà : aller sur le site web du langage Python et suivre les instructions d'installation pour votre système d'exploitation

Décompresser le fichier que vous avez téléchargé (au format .zip, .tgz ou .tar.bz2) dans un répertoire tel que ; se placer dans le répertoire Karrigell-2.3.5, nous l'appellerons le Répertoire Serveur

Pour lancer le programme, ouvrez une fenêtre console, mettez-vous dans le Répertoire Serveur et exécutez

(server dir)>Karrigell-2.3.5>python Karrigell.py
Vous devriez voir apparaître le message :
Karrigell 2.3.5 running on port 80
Debug level 1
Type Ctrl+C to stop

Entrez http://localhost dans le champ Adresse de votre navigateur : vous devriez voir la page d'accueil de Karrigell

Je conseille de définir dès l'installation l'identifiant et le mot de passe de l'administrateur : ils sont utilisés dans de nombreux programmes, dont ceux qui permettent de gérer des bases de données MySQL ou SQLite. Pour cela, ouvrez une autre fenêtre console, allez dans le dossier admin, puis exécutez le script k_password.py

(server dir)>Karrigell-2.3.5/admin>python k_password.py
et saisissez votre identifiant et mot de passe
