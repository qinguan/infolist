9. Scripts CGI

Nouveau en version 2.3.1

Si vous avez développé des scripts CGI dans un autre environnement, vous pouvez les exécuter avec Karrigell sans modification (il faut simplement que les scripts aient l'extension .py)

Tout ce qu'il y a à faire est de les placer dans des répertoires spécifiques. Par défaut, utilisez le dossier webapps/cgi-bin ; d'autres chemins peuvent être spécifiés dans le fichier de configuration, section Directories, option cgi

Avertissement : ceci ne fonctionne qu'avec le serveur asynchrone. La prise en compte des autres serveurs sera réalisée dans les prochaines versions
